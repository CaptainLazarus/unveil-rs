# Unveil Rs

## Directory structure

Unveil expect the following directory structure to render a presentation : 
```
-- project_dir
  |
  |__ unveil.toml 
  |__ slides 
  |  |
  |  |__ landing.md (default)
  |  |
  |  | 
  |  |__ slide-01.toml
```


## Command

|name | description      | 
| --- | -----------      | 
|init | new project      |
|build| build the projec |
|run  | serve the project|
